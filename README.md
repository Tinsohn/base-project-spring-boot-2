# ⭐️ Base Project ⭐️

Proyecto base con una estructura lista para crear proyectos API REST profesionales a partir de esta.

### Características 📋
* Java 17
* Spring Boot 2.5.5
* Spring Security
* Spring Web
* Spring Boot DevTools
* Spring Data JPA
* H2 Database
* Lombok
* MapStruct
* Swagger (Springfox)
* Autenticación con JWT
* UUID
* Manejo de roles ("ADMIN" y "USER" como base)
* Arquitectura en capas
